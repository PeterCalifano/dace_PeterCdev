Evaluate the compiled polynomial with a vector of any arithmetic type (such as DA or double) and return vector of results.

Parameters
[in]argsthe values of the independent DA variables to evaluate with. Must be a std::vector<> (or derived class) of an arithmetic type. If less than the number of independent DA variables defined during the DACE initialization are given, the missing entries are assumed to be zero.
